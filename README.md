# Cryptocurrency Price Prediction using LSTM

This project performs time series analysis and price prediction on cryptocurrencies
(Bitcoin and Ethereum) using LSTM neural networks.

## Features
- Historical price analysis
- LSTM-based prediction
- 30-day future forecasting
- Streamlit interactive dashboard
- BTC / ETH coin selector

## Tech Stack
- Python
- Pandas, NumPy
- TensorFlow / Keras
- Streamlit
- yfinance

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
